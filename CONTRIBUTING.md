# Contributing

This repository is intended to host public-facing documentation for MSMP.

## Suggested contributions
- Fix typos, broken links, or unclear instructions in `Docs/`.
- Propose wording changes via pull request.

## Not accepted here
- Source code changes (MSMP source is not hosted in this repository).
- License key requests or account/billing issues (email support instead).

## Support
Email `support@mythicreliquary.com`.
